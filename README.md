# Ranbir Kumar

AI/ML and robotics engineer building practical intelligent systems: RAG document assistants, autonomous field robotics environments, semantic search infrastructure, computer vision pipelines, and full-stack AI products.

I like working at the intersection of models, software systems, and real-world deployment. My projects usually start with a concrete problem, then move through data, backend architecture, interfaces, evaluation, and deployment until the system is usable by someone other than me.

## Focus

- AI/ML systems: RAG, semantic search, few-shot learning, explainability, applied prediction
- Robotics and autonomy: field robotics simulation, planning constraints, agent evaluation
- Computer vision: medical imaging, agriculture, detection/classification pipelines
- Full-stack engineering: FastAPI, Flask, Next.js, React, Supabase, PostgreSQL, Docker
- Product-minded engineering: demos, deployment, user workflows, and measurable outcomes
